Logback 의 MDC(Mapped Diagnostic Context) 를 이용하여, 요청의 흐름을 추적.
웹과 같은 환경에선, 여러 요청이 동시에 발생하기에, 특정 요청에 대한 추적이 힘듬.
이를 MDC 에 추적을 위한 값을 등록하여 해결.
단, 기본적으론 프로세스나 서버가 나뉘면 추적 못함.

사용방법은 java 의 Map 과 같이
put() 메소드로 값을 넣고, get() 메소드로 값을 받는다.
MDC에 등록한 값은 요청이 끝날때까지 유지됨.

매번 코드로 하기 번거로우니, 인터셉터를 이용하여 MDC 에 요청 ID 를 등록하고,
트래킹이 필요한 로그에서 등록한 값을 꺼내서 사용.
